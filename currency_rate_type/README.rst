==================
Currency Rate Type
==================

This module adds currency rate types to currency.

Installation
============

To install this module, you need to:

* put this module in a addons folder
* update the module list
* search for "Currency Rate Type" in your addons
* install the module

Configuration
=============

To configure this module, you need to:

* Go to Accounting -> Miscellaneous -> Currency Rate Types and define currency rate types

Usage
=====

This module is used by other modules which requires currency rate type.
